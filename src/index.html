інтернет-банкінг через улюблені Бухгалтерські програми
API від Altbank — це програмний інтерфейс, що надає можливість взаємодіяти з інтернет-банкінгом через звичні для вас бухгалтерські програми.

Через API від Altbank

ви зможете:

  1. Надсилати до банку платіжні документи (UAH), створені у вашій бухгалтерській програмі
  2. Отримувати баланс поточних рахунків у всіх валютах
  3. Налаштувати автоматичне отримання виписок у вашій бухгалтерській програмі
  4. Переглядати перелік транзакцій за поточний календарний день або за один із семи попередніх днів
  5. Отримувати курси валют та котирування товарів
