pinly - a jQuery Plugin extension for a numerical pin style entry.

Features of pinly:

  1. Compatible with desktop & mobile devices
  2. Call pinly.js as a jquery method e.g. $( this ).pinly();
  3. Number of inputs are configurable but pinly will default to 4 if unset
  4. pinly only accepts numbers, using first the browser to configure a data input with a js fallback
  5. pinly supports keyboard shortcuts: -
    • TAB or RIGHT ARROW moves to next input in the line
    • SHIFT or LEFT ARROW moves to the previous input in the line
    • UP increases the increment
    • DOWN decreases the increment
    • BACKSPACE moves the previous input to allow for corrections, on submission fail hitting backspace will reset the form
  6. pinly's behaviours, look and feel are 100% customisable using only CSS and minimal js
  7. pinly is Mobile friendly using the number pad on iOS devices while responsive CSS will fit pinly to your device
  8. Define a Custom Callback method to run when a pin has been entered
    • Use jQuery's $.trigger method to fire the pinly custom events pinlyPass & pinlyFail to define your prefered response for these actions
